# Stamped | https://usestamped.us/

**Stamped** is a secure and intelligent onboarding platform designed for financial institutions. It streamlines client and vendor onboarding with AI-powered features, multi-role dashboards, and comprehensive compliance tools.

## Features

### Core Functionality
- **Multi-Role Portals**: Separate dashboards for Clients, Relationship Managers, Compliance Officers, Risk Analysts, and Executives
- **Lead Management**: Full pipeline tracking from prospecting to onboarding with drag-and-drop Kanban board
- **Document Management**: Drag-and-drop uploads with automated validation and review workflows
- **AI-Powered Lead Scoring**: Intelligent lead qualification with detailed scoring breakdowns
- **Risk Assessment**: Comprehensive risk analysis with adverse media screening and sanctions checking
- **Real-time Notifications**: Instant updates across all user activities
- **AI Chatbot**: Context-aware assistant for client support

### Key Highlights
- **Professional Design**: Modern glassmorphism UI with smooth animations
- **Type-Safe**: Built entirely with TypeScript
- **Authentication**: Role-based access control with secure login flows
- **Mock Data**: Comprehensive demo data for immediate testing and presentations

## Tech Stack

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS, Framer Motion
- **Authentication**: Supabase Auth
- **UI Components**: Custom component library with Radix UI primitives
- **Forms**: React Hook Form with Zod validation
- **Charts**: Recharts

## Getting Started

### Prerequisites
- Node.js 20+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/nethum529/stamped.git
cd stamped

# Install dependencies
npm install

# Run development server
npm run dev
```

## User Roles

- **Client**: Access onboarding portal, upload documents, communicate with officers
- **Relationship Manager**: Manage leads, track pipeline, schedule meetings
- **Compliance Officer**: Review documents, assess risk, approve onboarding
- **Risk Analyst**: Analyze risk scores, generate reports, monitor threats
- **Executive**: View high-level metrics, strategic insights, and KPIs
